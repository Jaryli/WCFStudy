##关于WCF

 	作为微软分布式通信框架的集大成者，WCF可谓相当的重量级。
	因为工作和技术学习的需要，打算对WCF知识体系做一下细致探索。

 本项目主要包含以下知识点：

1. WCF服务寄宿单独
2. WCF服务绑定及探索
3. WCFRESTFUL实践
4. WCF服务实例管理
5. WCF可靠性会话
6. WCF行为解析
